# SOS Otica

Landing page institucional da SOS Otica, feita com Next.js para deploy na Vercel.

## Comandos

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Deploy na Vercel

- Framework preset: Next.js
- Build command: `npm run build`
- Install command: `npm install`
- Output directory: padrao do Next.js

Os assets publicos ficam em `public/` e o codigo da aplicacao fica em `src/`.
